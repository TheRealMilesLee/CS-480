# CS-480

CS-480 is a project repository focused on implementing various search algorithms and puzzles, primarily using Python. It includes a collection of scripts and examples for educational and research purposes in artificial intelligence and algorithm design.

## Features and Functionality

- Implementation of classic search algorithms such as BFS, DFS, and A*.
- Solving puzzles like the 8-puzzle and River Cross.
- Support for informed search techniques with priority queues.
- Test cases and examples for algorithm validation and performance analysis.

# CS-480 - AI Algorithm Implementation Project

CS-480 是一个专注于人工智能算法实现的项目，包含多种经典问题的求解方案，如八数码问题、河流过桥问题等。项目结合了Python脚本和iOS/macOS开发框架，支持算法演示与交互式调试。

## 功能特性
- 八数码问题（8-Puzzle）求解器
- 河流过桥问题（River Cross）模拟器
- 启发式搜索算法实现（A*、IDA*）
- 状态空间搜索算法（BFS、DFS）
- 可视化交互界面（iOS/macOS）
- 算法性能分析工具
